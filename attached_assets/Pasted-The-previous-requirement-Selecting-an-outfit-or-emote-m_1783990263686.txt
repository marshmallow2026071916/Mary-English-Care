Official Unlock State
--------------------------------------------------

Unlocked:
- outfit_idle_001.png

- outfit_smile_001.png

Selection must NOT
--------------------------------------------------

Selecting an outfit or an emote must NEVER:

- modify the unlock state
- recalculate the unlock state
- reset the unlock state
- overwrite the unlock state
- filter the unlocked list
- rebuild the unlocked list based on the current selection
