--------------------------------------------------
Official behavior
--------------------------------------------------

The small icon on each Review Log conversation card represents the outfit that belonged to that Review Log level.

It does NOT represent Mary's current appearance.

It must NOT depend on:

- selectedOutfit
- selectedEmote
- selectedReviewReward
- Current Appearance
- Current Appearance Icon

Instead, it must use the Review Log's own saved level information.

Use:

- reviewLog.level
or
- reviewLog.levelOutfit

to determine which outfit icon to display.

--------------------------------------------------
Mapping
--------------------------------------------------

Level 0
or
reviewLog.levelOutfit = outfit_000

→ icon_outfit_000.png


Levels 1–5
or
reviewLog.levelOutfit = outfit_001

→ icon_outfit_001.png


Levels 6–10
or
reviewLog.levelOutfit = outfit_002

→ icon_outfit_002.png


Continue using the same five-level pattern.

Even if the player is currently wearing another outfit or currently displaying a Review Reward, these icons must remain unchanged because they represent the outfit of that saved Review Log.
