==================================================

Replace the current Correction card.

A)

🌟 Excellent!

Use when Mary especially praises the user's English.

Card title:

🌟 Excellent!

Body:

Short praise message only.

--------------------------------------------------

B)

✅ Perfect!

Use when the sentence is already natural.

Card title:

✅ Perfect!

Body:

Perfect!

--------------------------------------------------

C)

💡 Mary's Suggestion

Use when the user's sentence is correct but a more natural expression is suggested.

Card title:

Display ONLY the suggested sentence.

--------------------------------------------------

D)

✏️ Mary's Correction

Use when the user's sentence needs correction.

Card title:

✏️ Mary's Correction

Body:

Display ONLY the corrected sentence.

Do NOT display the user's original sentence.
