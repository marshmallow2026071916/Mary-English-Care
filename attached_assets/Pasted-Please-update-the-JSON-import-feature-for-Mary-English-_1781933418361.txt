Import behavior:

- Parse the JSON.
- Validate version.
- For now, support version: 2.
- Validate that the JSON contains date, progress, and reviewLog.
- Save progress data to localStorage.
- Save reviewLog data to localStorage.
- If the same date already exists, replace the existing data.
- Update the Top screen using progress.
- Update the Tasks screen using progress.
- Update the Review Log screen using reviewLog.

Startup behavior:

- When Mary English starts, automatically load the latest saved JSON from localStorage.
- Restore progress from progress.
- Restore Review Log from saved reviewLog data.
- The user should not need to import the same JSON twice.

Important:

- The user imports only one JSON per day.
- Do not require separate imports for progress and reviewLog.
- Treat the latest valid version 2 JSON as the source of truth.
- Keep old progress-only JSON support only if it already exists, but the official format is now version 2.
- Do not change unrelated functionality.
